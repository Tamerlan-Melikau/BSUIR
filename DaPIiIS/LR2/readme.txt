Экспертная система: диагностика по симптомам

Простая экспертная система на базе библиотеки `experta`.

Где можно использовать Experta:

Experta подходит для создания небольших экспертных систем, систем поддержки принятия решений, диагностических приложений, чат-ботов с логикой, обучающих программ по искусственному интеллекту. Библиотека позволяет описывать правила в декларативном стиле и удобна для быстрого прототипирования.
