# Relations Operations in C#

A C# console application for working with binary relations represented as pairs of integers.

## Features

The application can:

* Split input data into ordered pairs (relations)
* Display relations as tuples
* Find the projection of a relation on the X set
* Find the projection of a relation on the Y set
* Invert relations (swap elements in each pair)
* Compose two relations (relation composition)
* Validate user input and ensure relations contain an even number of elements
