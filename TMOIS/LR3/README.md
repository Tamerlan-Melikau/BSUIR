# Binary Relations Operations in C#

A C# console application for working with binary relations and set operations.

## Features

The application can:

* Perform union of relations (Union)
* Perform intersection of relations (Intersection)
* Find the difference between relations (Difference)
* Invert relations (Inverse Relation)
* Compose two relations (Relation Composition)
* Calculate the image of a subset under a relation
* Calculate the preimage of a subset under a relation
* Work with user-defined sets and binary relations
* Validate and process input data interactively
