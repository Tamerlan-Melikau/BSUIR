# Doubly Linked List Operations in C++

A C++ console application that implements a doubly linked list and provides various operations for managing and processing list elements.

## Features

* Create a new doubly linked list
* Display list elements from beginning to end
* Display list elements in reverse order
* Add elements to the beginning of the list
* Add elements to the end of the list
* Calculate the average value of all elements
* Remove elements smaller than the average value
* Dynamic memory management using doubly linked lists
* Interactive menu-driven interface

## Data Structure

The application uses a doubly linked list where each node contains:

* Data value
* Pointer to the next node
* Pointer to the previous node

## Purpose

This project demonstrates the implementation of doubly linked lists, bidirectional traversal, dynamic memory allocation, and basic list processing algorithms in C++.
