# Maximum Element Search in C++

A C++ program that finds the maximum element in an array using two different approaches.

## Features

* Finds the maximum element recursively using the divide-and-conquer method
* Finds the maximum element iteratively using a loop
* Dynamically allocates memory for the array
* Validates user input
* Compares recursive and non-recursive solutions

## Algorithms

### Recursive Approach

The array is divided into two halves recursively until a single element remains. The maximum values from the left and right halves are then compared and returned.

### Iterative Approach

The array is traversed sequentially, keeping track of the largest element found.

## Purpose

This project demonstrates the implementation and comparison of recursive and iterative algorithms for solving the same problem.
