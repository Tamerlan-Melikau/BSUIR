# Binary Search Tree Directory in C++

A C++ console application that implements a Binary Search Tree (BST) for storing and managing records consisting of numeric keys and associated names.

## Features

* Create and maintain a binary search tree
* Add new records with a unique key and name
* Search for records by key
* Update existing records
* Delete records by key
* Display the tree structure visually
* Traverse the tree using:

  * Pre-order traversal
  * In-order traversal
  * Post-order traversal
* Count nodes that have two children
* Interactive menu-driven interface

## Data Structure

The application uses a Binary Search Tree where each node contains:

* Integer key
* Associated name
* Pointer to the left child
* Pointer to the right child

## Algorithms

### Search

Efficiently locates records using BST properties.

### Insertion

Adds new nodes while maintaining the binary search tree structure.

### Deletion

Removes nodes while preserving BST ordering rules.

### Tree Traversal

Supports pre-order, in-order, and post-order traversals for tree exploration and data output.
