# Граф: матрица инцидентности → матрица смежности

Программа преобразует матрицу инцидентности в матрицу смежности для **неориентированного графа**.

## Как работает

1. Пользователь вводит количество вершин и количество рёбер.
2. Заполняет матрицу инцидентности (строки — вершины, столбцы — рёбра).
3. Для каждого столбца (ребра) находит две вершины, содержащие `1`.
4. Добавляет ребро в матрицу смежности, проставляя `1` на соответствующих позициях.
5. Выводит полученную матрицу смежности.
