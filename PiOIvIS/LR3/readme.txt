Логический вывод: расчёт премии сотрудников

Программа на основе `kanren` — реляционного/логического программирования на Python.

Где можно использовать Kanren:

Kanren подходит для задач логического вывода, символьных вычислений, проверки выполнимости условий, построения экспертных систем, систем вопросов-ответов, генерации тестовых данных и верификации бизнес-правил. Позволяет описывать отношения между данными и находить все решения, удовлетворяющие ограничениям.
