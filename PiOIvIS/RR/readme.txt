Алгоритм Диница (максимальный поток)

Реализация алгоритма Диница для поиска максимального потока в ориентированном графе.  
Представление графа — список инцидентности.

Где можно использовать алгоритм максимального потока:

Алгоритмы поиска максимального потока применяются в логистике (оптимизация грузоперевозок), планировании производственных цепочек, распределении сетевого трафика, анализе транспортных сетей, компьютерном зрении (сегментация изображений), распределении задач в кластерах, моделировании жидкостей в трубопроводах.
